WAO Let's code Py のとは？

  WAO Let's Code Py とは京都大学発のITベンチャー
DeMiAと協力して開発した受験生のためのアプリです。
WAO Let's Code Py では一問一答形式の問題やPythonを
実行できるエディターアプリ、情報Ⅰ・共通テスト予想問
題を利用できます。

WAO Let's code Py の魅力！

１．一問一答形式の問題

   共通テスト・情報Ⅰの各章ごとの問題があり、
全578問収録されています。ワンタッチで答え
をチェックすることができます。自分の苦手分
野を徹底的に勉強することが可能です。

２．エディター

   環境整備をすることなく、このアプリ一つ,
でPythonのコードを実行することができます。
問題を解くだけでなく実際にコードを実行して
アウトプットすることで本当の理解に繋がりま
す。文科省のサンプルコード準拠です。

３．予想問題

   独自に作成した共通テスト予想問題が利用
できます。質が高く本番と同じ形式の問題を
こなすことで自分なりの解き方や時間配分が
分かるようになります。問題は随時追加予定
のため問題数に困ることはありません。
